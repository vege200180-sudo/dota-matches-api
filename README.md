# Dota 2 Matches API

API для получения предстоящих матчей Dota 2 с Liquipedia и данных команд с OpenDota.

## Endpoints

- `GET /api` - Информация об API
- `GET /api/matches` - Предстоящие матчи с Liquipedia
- `GET /api/teams` - Топ команды с OpenDota
- `GET /api/teams/{id}/players` - Игроки команды
